Evolutionary Path

Project Changelog.

Tracking the architectural development and structural refinement of the Advet Buildwell digital presence.

v1.2.0

The Advisory Phase

March 13, 2026

Analytics & Intelligence

  • Admin Analytics Portal

    Launched a dedicated Analytics page (`admin-analytics.html`) featuring demographic doughnuts, conversion funnels, and regional demand data grids using custom-tailored Chart.js integrations.

  • Market Frequency Dashboard

    Integrated a sophisticated Valuation Growth line chart into the central Admin Dashboard, alongside new 'Asset Health' optimization trackers.

Governance & Settings

  • Studio Settings Portal

    Released `admin-settings.html` to centralize platform configuration. Allows real-time toggling of 'Atmospheric Reveal' animations, MFA security policies, and newsletter communication rhythms.

Curation Engine Updates

  • Atmospheric Property Inputs

    Updated the `add-property.html` flow to capture nuanced architectural data, including "Sensory Layers" (e.g., Morning Light) and Material Palettes, synchronizing perfectly with the consumer-facing detail pages.

  • Structural Stability Fixes

    Cured gallery overlapping bugs in property detail views by switching from fixed heights to responsive aspect ratios (`aspect-video`, `aspect-[4/3]`) and optimized the global Admin sidebar width (`w-80`) across all portals.

v1.1.0

The Refinement Phase

March 13, 2026

Asset Localization

  • Offline-Ready Architecture

    Migrated all core CSS and JavaScript dependencies from external CDNs to the local `assets/` directory. This includes Tailwind CSS and Google Fonts (DM Sans, Cormorant Garamond).

  • Local Image Preservation

    Began transitioning from Unsplash API to locally stored high-fidelity imagery, starting with the "Sanctuary 04" listing.

Global Enhancements

  • Field Notes Newsletter

    Integrated a standardized newsletter subscription feature into the brand footer across all 12+ public and utility pages.

  • Unified Footer Design

    Refined the global footer to a 12-column grid system, ensuring perfect alignment of pages, social links, and studio contact information.

Admin Core Updates

  • Standardized Admin Sidebar

    Unified the navigation patterns for internal tools, ensuring a consistent experience across the Dashboard, Listing Management, and Inquiry Portals.
